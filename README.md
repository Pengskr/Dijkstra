# 基于Matlab的Dijkstra算法

参考：<https://www.bilibili.com/video/BV19T4y1M7uR/?spm_id_from=333.788.recommend_more_video.0&vd_source=be5bd51fafff7d21180e251563899e5e>

**Dijkstra算法**:从一个节点遍历 *其余各节点的* 最短路径算法，解决的是有权图中最短路径问题。
如果仅仅只是想找到 *起点到终点* 的最短距离以及路径，那么在将终点从U集移动到S集后算法即可结束。

Dijkstra_demo.mlx 使用Matlab实时脚本演示算法。
